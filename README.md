# anomalib-workshop
Anomalibワークショップの教材です

# セットアップ
```bash
pip install -r requirements.txt
python -m ipykernel install --name anomalib --display-name "anomalib"
```

## データセットダウンロード
https://github.com/cm-koga/assets/releases/download/workshop/wood.zip

ダウンロードしたwood.zipをnoteboob/datasets/の下で解凍

## 起動
```bash
jupyter lab
```

## カーネルの選択
jupyterのカーネルは「anomalib」を選択してください
